Editing a course

As a staff member, you can only edit courses that your Pocket Centre has created.

  • To edit a course, select the course in the Courses page. You should see an editable version of the course profile.

  • Listing status

    By default, a course is set to Unlisted. An unlisted status means the course can't be seen by public. A listed course means the public can view and register for the course (if course bookings were created). To change the listing status of a course, toggle the Listed field.

  • Course flyer

    A course profile can only display 1 flyer at a time.

  • Important:

    If a course is listed, any changes you make are displayed immediately to public upon saving.
