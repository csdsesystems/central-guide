Repeat a booking

The repeat function is a quick way of duplicating a booking based on a frequency and occurence pattern that you specify.

  • To repeat a booking

    Navigate to the booking you would like to repeat. Click Actions dropdown and then click Repeat.

  • In this popup, specify the frequency and the number of occurences. Note that the dates will be relative to the date of the booking you are repeating.

    For example, if you have the same booking every week for the next 5 weeks. Specify the frequency first. In this case, the booking repeats every 1 week...

    ...for the next 5 weeks.

    Click Preview to see the dates for the new bookings. If you're happy with the dates, click Repeat. The bookings will be created in the background for you.

    Take note that this function only creates copies of the booking you are currently in. If you make a change to any of the bookings created in this process, it won't apply to the rest of the bookings. Repeated bookings are not currently linked or synced in any way.
