Adding and updating rooms to a booking

A course booking allows you to run a course from start to finish. If you haven't created one yet, learn how to create a booking.

  • To add a room, click Add room.

    Fill out the following information for every room. Click Save to update the booking.

    In specifying the time for the room, you can include the preparation and pack-up time. This will only affect the room availability and will not change the booking time.

    What do I do if I don't see any venue or rooms? Learn how to configure your Pocket Centre settings.

    To update/remove rooms from the booking, click Edit.
