Adding and updating rooms to a booking

A course booking profile allows you to run a course from start to finish. If you haven't created one yet, learn how to create a booking.

  • To manage rooms your Pocket Centre should have been configured already. If you can't find any rooms in the booking form, learn how to add rooms to your Pocket Centre.

    To add a room, click Add room.

    Fill out the following information for every room. Click Save to update the booking.

    In specifying the time for the room, you can include the preparation and pack-up time. This will not change the booking time, but will only reflect on the room availability.

    What do I do if I can't see any venue or rooms? Learn how to configure Bookings section.

    To update/remove rooms from the booking, click Edit.
