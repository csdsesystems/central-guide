User accounts

As part of the CSDS Pocket Centre Network, you also have access to view user registration and records. This will assist you in providing support to other staff in your pocket and to your course attendees.

Important: If you need to update any information in a user's account, please contact us.

  • A user account profile shows you (1) basic information about a user which includes postal address, professional information and emergency contact, (2) user roles (i.e. facilitator or faculty), (3) course registrations, (4) payments, (5) equipment on loan, (6) eLearning groups, (7) Pocket Centres, (8) VOCs, and (9) system emails.

  • You can find the Accounts menu item in the left sidebar. Click Accounts. Type in the name or email address of the user you want to search for and hit enter.

    Click on the row to see a preview of the user account. Click on the user's name to go to the full user account profile.
